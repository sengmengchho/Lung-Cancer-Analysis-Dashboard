# Lung Cancer Analysis Dashboard

## 📊 Project Overview

This project contains a comprehensive data visualization dashboard analyzing lung cancer statistics and trends. The dashboard is built using Power BI and provides insights into various aspects of lung cancer data through interactive visualizations.

## 📁 Project Contents

- **Lung Cancer.pbix** - Power BI dashboard file containing interactive visualizations and analyses
- **lung_cancer_Dataset.xlsx** - Dataset containing lung cancer statistics and related metrics
- **README.md** - Project documentation (this file)

## 📈 Dataset Description

The `lung_cancer_Dataset.xlsx` file contains lung cancer data with various attributes including:
- Patient demographics
- Cancer statistics
- Risk factors
- Treatment information
- Survival rates
- And other relevant health metrics

**Note:** Ensure the dataset is properly loaded into Power BI for the dashboard to function correctly.

## 🎯 Dashboard Features

The Lung Cancer Analysis Dashboard includes:
- **Interactive Visualizations** - Charts, graphs, and maps for data exploration
- **Key Metrics** - Important statistics and KPIs related to lung cancer
- **Filtering Capabilities** - Dynamic filters for analyzing specific data segments
- **Trends Analysis** - Time-series visualizations showing cancer trends
- **Demographic Breakdowns** - Analysis by age, gender, location, and other factors

## 🚀 How to Use

1. **Open the Dashboard**
   - Open `Lung Cancer.pbix` in Microsoft Power BI Desktop
   
2. **Connect Data**
   - Ensure `lung_cancer_Dataset.xlsx` is properly linked to the dashboard
   - Refresh data if needed from the Power BI ribbon

3. **Explore Visualizations**
   - Interact with charts and graphs to explore the data
   - Use filters to drill down into specific segments
   - Hover over data points for detailed information

4. **Export Reports**
   - Share insights by exporting visualizations
   - Create custom views by applying filters

## 🛠️ Requirements

- **Microsoft Power BI Desktop** (latest version recommended)
- **Microsoft Excel** - For viewing/editing the dataset
- **Data Connectivity** - Ensure proper file paths are configured
